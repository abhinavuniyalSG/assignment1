## Sementic tags used 
# header 
It represents  introductory section of the page. 
It contains the page title and logo.

#   main 
It  represents the focus content. 
one main element per page.

# article 
It represents a self-contained part of  content. 
card is an article because it can stand on its own.

# h1 
It represents the main heading of the page. 
One h1 per page.

# h2 
It represents a second-level heading. Used for the names of the team members.

# p 
It represents paragraphs of text such as the job title and description.

# img
To add images and it uses the alt attribute to provide alternative text for accessibility.

# a 
Used for hyperlinks in social media profiles.


## CSS Properties and Concepts Used
# CSS Grid 
Used to create the responsive layout for the main content.
Our main section uses the grid to arrange cards.

# Flexbox
Used to align elements within containers, like centering content and arranging items horizontally or vertically.
Our header uses the flex to align div in a row.

# Media Queries 
Used to make layout responsive by adjusting the design according to screen sizes.
Configure our screen size according to desktop , tablet & mobile view.
